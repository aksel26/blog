Gatsby React 를 활용해서 블로그 페이지를 만들어줘.

style은 tailwindcss v4기준으로 할거야.

예시 블로그 파일들을 만들어줘.

페이지 구성은 아래처럼 할거야.

- **기술 블로그**
  - 코드 가독성, 정확성, 재현 가능성이 중요
- **일상 블로그**
  - 생생한 사진, 솔직한 후기, 실용적인 정보가 중요
